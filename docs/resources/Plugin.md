本节教程演示如何生成一个解决方案, 编译插件并加载到游戏里获得日志反馈.  

## 解决方案

按照[工具配置](/docs/setup/Setup.md)搭建好`SKSEPlugins`开发环境并完成必需的`BOOTSTRAP`步骤后, 使用`!MakeNew MyNewPlugin`脚本生成一个名为`MyNewPlugin`的插件项目, 随后通过`!Rebuild`脚本生成解决方案并预编译`CLib`静态库, 以节省后期编译的时间.
